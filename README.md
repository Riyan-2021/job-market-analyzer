
# Job Market Analyzer

## Overview
This project scrapes job listings and analyzes trending skills in the job market.

## Features
- Web scraping of job listings
- Skill frequency analysis
- Dashboard visualization using Streamlit

## Technologies Used
- Python
- BeautifulSoup
- Pandas
- Matplotlib
- Streamlit

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run analysis:
python src/analysis.py

3. Run dashboard:
streamlit run app/app.py

## Dataset
Sample dataset included in data/sample_jobs.csv
